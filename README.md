# 企业知识库decay监控

知识库内容过时没人更新？
**监控帮你发现老化内容。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 知识不再"过时"

- **老化检测**：识别过时/失效的知识条目
- **衰退趋势**：时间维度可视化
- **维护建议**：自动生成更新计划
- **健康评分**：知识库整体健康度

## 怎么用

打开HTML文件 → 添加知识条目 → 查看健康报告
